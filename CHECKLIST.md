# Build Quality Checklist

## Pre-Build
- [ ] BIOS compatibility verified
- [ ] PSU wattage and connector coverage checked
- [ ] Cooler clearance and case dimensions verified
- [ ] RAM QVL/support checked
- [ ] Storage plan defined (OS and data layout)

## Assembly
- [ ] CPU seated and secured correctly
- [ ] Thermal paste applied correctly
- [ ] Cooler mounted with even pressure
- [ ] Front panel and fan headers connected
- [ ] Cable routing does not block airflow

## First Boot
- [ ] POST successful
- [ ] BIOS updated
- [ ] Memory profile configured (if stable)
- [ ] Boot order configured correctly
- [ ] Baseline temperatures checked in BIOS

## Operating System Setup
- [ ] Clean OS install completed
- [ ] Chipset, GPU, and network drivers installed
- [ ] Security updates completed
- [ ] Monitoring tools installed

## Validation
- [ ] Idle temperatures recorded
- [ ] Load temperatures recorded
- [ ] CPU stress test passed
- [ ] Memory test passed
- [ ] Storage test passed
- [ ] No crashes under mixed workload

## Finalization
- [ ] Stable profile documented in BENCHMARKS.md
- [ ] Build timeline updated in BUILD_LOG.md
- [ ] Next upgrade targets documented in ROADMAP.md
